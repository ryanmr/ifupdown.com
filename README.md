# ifupdown.com

Landing page for [ifupdown.com](https://ifupdown.com). A single self-contained HTML file with no external dependencies.

## Overview

- Inline CSS, SVG, and JavaScript — zero external requests
- Dark mode via `prefers-color-scheme`
- Canvas particle animation around the logo
- Responsive, system font stack

## Deployment

Copy `index.html` to your web server. That's it.
